🧠 BrainDigits — Multiplayer Number Guessing Game

Play here 👉 https://braindigits.vercel.app

BrainDigits is a fun and interactive online multiplayer number guessing game where players compete in real-time to guess a hidden number generated by the system. It’s simple, fast, and perfect for playing with friends.

🚀 Features
🎮 Multiplayer Gameplay
Create or join rooms using a unique room code and play with friends.
🔢 Random Number Generation
The system generates a hidden number within a fixed range (e.g., 1–100).
🔁 Turn-Based Guessing
Players take turns to guess the number.
💡 Smart Hints
Get instant feedback like “Too High” or “Too Low” after each guess.
🏆 Winner Detection
First player to guess correctly wins the game.
⚡ Fast & Lightweight UI
Smooth gameplay with minimal loading time.
